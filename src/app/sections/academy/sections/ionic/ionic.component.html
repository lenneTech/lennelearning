Installation
Um mithilfe von Ionic hybride mobile Applikationen implementieren zu können, musst du vorher einige Pakete installieren.
Dadurch, dass du an dieser Stelle schon mit nodeJS, npm und Angular gearbeitet haben solltest, setzen wir voraus, dass du diese Pakete bereits installiert hast. Damit deine Ionic Projekte auf nativen Mobilgeräten installiert werden können, müssen sie für die einzelnen Plattformen kompiliert werden. Dazu benötigst du Android Studio für Android Applikationen und XCode für IOS Applikationen. Wenn du auf Windows entwickelst, kannst du XCode leider nicht verwenden. Teste deine Anwendung dann erstmal für Android Geräte. Nachdem du Android Studio und/oder XCode erfolgreich installiert hast, gehe wie folgt vor:
Öffne Android Studio, klicke unten rechts auf Configure und wähle SDK Manager aus. Hier können alle SDK (Softwware Development Kit) Versionen eingesehen, installiert oder deinstalliert werden. Installiere am besten verschiedene SDKs mit einem Api Level von mindestens 21. Je höher das Api Level desto aktueller ist die Version des SDKs.
Darufhin kannst du die Ionic CLI mit npm install -g @ionic/cli installieren
Projekt
Erstelle dein eigenes Projekt wie folgt:
Führe ionic start aus, um dein eigenes Projekt zu initialisieren
Wähle hier angular aus, um in deinem Projekt mit dem Angular Framework zu arbeiten
Gib dem Projekt einen Namen
Als Nächstes solltest du zwischen verschiedenen Starter Optionen auswählen können. Wähle hier den Tabs Starter aus, um mit einem Template zu starten, das schon eine Tabbar mit Navigation implementiert hat
Bei der Frage, ob du Capacitor einbinden möchtest, solltest du n eingeben, da wir diesen Schritt gleich manuell übernehmen
Mit ionic serve solltest du jetzt schon deine Anwendung im Browser starten können
Für das Enbinden von nativen Funktionen verwenden wir Capacitor. Um es in deine erstellte Anwendung einzubinden führe folgende Befehle in deinem Projekt aus:
npm install @capacitor/core @capacitor/cli
npx cap init
Android
Um deine App auf Android Geräten installieren zu können, gehe wie folgt vor:
npm install @capacitor/android installiert das Paket
npx cap add android erstellt Android als Platform in deinem Projekt und erstellt einen Android Ordner
npx cap open android öffnet den Android Ordner in Android Studio
Mit ionic build Wird dein Projekt kompiliert im www Ordner kompiliert
Führe danach immer npx cap sync aus, um den aktuellen Build auch auf die nativen Plattformen zu übertragen.
Führe ionic capacitor run android -l --external in deinem Projekt aus, um die Anwendung auf einem Android Emulator zu installieren und auszuführen.
Durch -l wird live reloading aktiviert, sodass bei Änderungen im Code die App automatisch neu kompiliert wird und --external öffnet automatisch XCode und einen Emulator. Falls der Emulator sich nicht automatisch öffnen sollte, kannst du im AVD Manager von Android Studio Emulatoren auch manuell hinzufügen, bearbeiten und starten.
