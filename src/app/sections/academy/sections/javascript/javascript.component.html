Quiz
Entwickle ein Quiz basierend auf den theoretischen Fragen, die du zu JavaScript in der letzten Zeit beantwortet hast. Du kannst natürlich auch ein Quiz für eine anderes Thema (z.B. ein Tierquiz) erstellen.
Gib dem Nutzer eine Wahlmöglichkeit von 4 Antworten pro Frage.
Die einzelnen Fragen sollen Punkte geben (z.B. jede richtige Antwort +2 und jede falsche -1).
Überlege dir eine Gewichtung der Fragen. Vielleicht gibt die ein oder andere Frage mehr Punkte, weil sie schwieriger ist.
Die richtigen bzw falschen Antworten sollen nach der Abgabe entweder grün oder rot gekennzeichnet werden.
Am Ende soll der Nutzer eine Endpunktzahl bekommen, die, je nach ihrer Größe, anders farbig markiert ist.
