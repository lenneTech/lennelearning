Projekt
Die Privatbrauerei Lennetaler braucht eine neue Website. Zunächst soll die Website rein gestalterisch mit Bootstrap umgesetzt werden. Folgende Anforderungen werden gestellt:
Eine Navbar mit Logo und Name, die Navigationsmöglichkeiten zur „Startseite“, „Shop“ und „Kontakt“, eine Suchzeile, um die Website (nicht funktional) nach Inhalten durchsuchen zu können und einen Warenkorb, der sich ausklappen lässt und ausgewählte Produkte beinhaltet.
Eine Startseite, die den Nutzer mit einem Begrüßungstext Willkommen heißt (Nutze dazu z.B. einen „Jumbotron“).
Eine Shopseite, mit einer Diashow von unternehmensspezifischen Bildern on top und einer Auflistung der verschiedenen Biersorten (Lennetaler „Eisenglanz“, „Bernstein“ und „Goldberg“) mit Hilfe von „Cards“. Die einzelnen Karten haben ein Bild, den Produktnamen und einen „Mehr Anzeigen“-Button, der die Kartendetails ausfahren lässt. „Goldberg“ ist neu im Sortiment und soll dementsprechend gekennzeichnet werden.
Eine Kontaktseite, die einerseits die Kontaktdaten von Lennetaler mit einer „List Group“ anzeigt und zudem ein Kontaktformular zur Buchung von Brauereibesichtigungen (E-Mail, Name, Vorname, Teilnehmeranzahl, Nachricht) beinhaltet. Nutze dafür die Bootstrap Validierung.
