Hochladen
Jetzt bist du an der Reihe mit Docker ein wenig herumzuspielen. Was du dafür machen musst:
Installiere Docker Desktop, sofern noch nicht geschehen.
Führe in deiner Konsole docker run -dp 80:80 docker/getting-started aus.
Öffne im Browser http://localhost und folge den Anweisungen der Seite.
