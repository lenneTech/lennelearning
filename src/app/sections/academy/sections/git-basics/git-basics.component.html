Inhalt
Wenn du ein eigenes Projekt auf die Beine stellst, liegt es zunächst nur auf deinem Computer und somit bist du der Einzige, dem der Quellcode vorliegt. Jetzt kommt Git ins Spiel. Mit Git kannst du für dein Projekt Back-ups erstellen, zu denen du jederzeit Zugriff hast und diese sogar online stellen. Wenn sich also ein Fehler in deinen Code einschleicht, kannst du einfach zu einer früheren Version deines Projekts zurückkehren und dort weitermachen. Wie das Ganze funktioniert, erklären wir dir jetzt.

Hochladen
Jetzt bist du an der Reihe, dein eigenes Projekt in GitHub hochzuladen. Befolge dabei diesen Leitfaden:
Lade dir Git herunter und installiere es.
Öffne Git Bash.
Mit dem Befehl cd [Ordnername]' bzw. 'cd .. wechselst du in der Konsole die Ordner. Gehe in den Ordner, in dem dein Projekt liegt.
Führe git init aus.
Führe git add . aus.
Führe git commit -m "deine Nachricht" aus.
Erstelle dir einen GitHub-Account, sofern das noch nicht geschehen ist.
Erstelle ein neues Projekt in GitHub, welches den Namen deines eigenen Projekts trägt.
Führe git remote add origin git@GitHub.com:[DeinName]/[DeinProjekt].git aus, um deinen Ordner mit dem git Repository zu verknüpfen.
Führe git branch -M main aus.
Führe git push -u origin main aus, um alle Daten aus deinem Ordner nach GitHub hochzuladen.
Wenn alle Dateien in deinem GitHub-Repository angezeigt werden, hast du alles richtig gemacht!
Herunterladen
Dein Projekt sollte jetzt bei GitHub vorliegen. Nun möchtest du es von z. B. einem anderen PC aus herunterladen. Wir imitieren das Ganze mit einem anderen Ordner.
Erstelle einen neuen Ordner und führe git init aus.
Führe git remote add origin git@GitHub.com:[DeinName]/[DeinProjekt].git aus, um deinen Ordner mit dem git Repository zu verknüpfen.
Führe git fetch aus, um zu prüfen, ob Daten, die nicht in deinem Ordner sind, auf GitHub liegen.
Führe git pull aus, um die Daten, die nicht in deinem Ordner sind, von GitHub herunterzuladen.
Eventuell musst du noch den Branch wechseln. Das geht mit git branch main .
Wenn alle Dateien im neu erstellten Ordner angezeigt werden, hat alles funktioniert.
