Was sind Mockups?
Wenn Du eine Idee für ein Projekt hast, ist es häufig schwierig andere verbal davon zu überzeugen. Meist hilft da eine visuelle Unterstützung, die Dein Vorhaben in den Grundzügen darstellt. Mockups sind digitale Visualisierungen von Ideen und Konzepten oder anders gesagt, eine Art Attrappe des zu gestaltenden Produktes in einer realistischen Umgebung.
Sie werden digital erstellt und dienen zu
Präsentationszwecken
der Absprache mit Auftraggeber:innen und Teammitgliedern
dem Einholen von Feedback.

Sie können in ihrem Detailgrad variieren, entsprechen aber meistens schon dem „Look and Feel“ des zu gestaltenden Produkts. Wenn es an die Finalisierung eines Projekts geht, dienen die Mockups häufig als Kommunikationsmittel zwischen Designer:innen und Entwickler:innen.
Mockups erstellen
Es gibt verschiedene Wege ein Mockup zu erstellen. Der schnellste und einfachste Weg ist die Verwendung von Mockup-Generatoren. Dabei wird das Design einfach als Bild in das Umgebungsmodell hochgeladen. Der Mockup- Generator platziert das Design automatisch und perspektivisch korrekt in das Umgebungsmodell. Anschließend kann das Mockup heruntergeladen und präsentiert werden. Einer dieser Mockup-Generatoren ist Smartmockups. In diesem Video wird erklärt, wie man ihn verwendet. CSS Lebenslauf
Verändere Deinen Lebenslauf mit CSS. Verwende dazu mindestens einmal alle Möglichkeiten, CSS einzubinden und HTML Elemente zu verändern. Gehe dabei wie folgt vor:
Verändere den Hintergrund, indem Du z. B. eine Farbe oder ein Bild festlegst.
Die HTML Standard Font ist ziemlich langweilig. Suche nach einer modernen Schrift und binde sie in Deinen Lebenslauf ein. Verschiedene Fonts findest Du unter anderem hier.
Hebe die Überschrift noch mehr hervor, indem Du ihr z. B. eine andere Farbe, Größe oder Ähnliches gibst.
Verändere Dein Bild, indem Du z. B. die Größe oder Border anpasst, Margin oder Padding einfügst.
Vereinheitliche die Farbe oder Größe von kursiv oder fettgedruckten Texten, indem Du dessen Selektoren gruppierst und so die entsprechenden Attribute änderst.
Hebe den jeweils ersten Stichpunkt der beiden Listen hervor, indem Du dessen Farbe oder Ähnliches änderst.
Verändere mithilfe von Pseudoklassen z. B. die Hintergrundfarbe eines Inputfeldes, wenn der Nutzer darüber „hovert“.
Zuletzt kannst Du den Hintergrund vom Footer ändern und den Unterstrich im Link entfernen.

Prettier
Du kannst inzwischen eigene Websites mit HTML und CSS erstellen. Übung gehört dazu, keine Frage. Aber die Grundlagen kannst Du nun. Je umfangreicher Deine Website und je anspruchsvoller Dein Styling wird, umso schneller kann Dein Code unübersichtlich werden. Struktur im Code ist das absolute A und O eines erfolgreichen Projekts. Wir möchten an dieser Stelle exemplarisch "Prettier" vorstellen.
Prettier ist eine Extension für Deinen Editor, die Dir einen Großteil der Formatierungsarbeit abnimmt. Durch einen Input wird Dein Code durchforstet, automatisiert formatiert und, nunja, prettier gemacht. Innerhalb der Einstellungen von Prettier findest Du alle Anpassungsmöglichkeiten. Speziell die "format on save" Funktion möchten wir Dir sehr ans Herz legen.
