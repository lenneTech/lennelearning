Was sind Mockups?
Wenn du eine Idee für ein Projekt hast, ist es häufig schwierig, andere verbal davon zu überzeugen. Meist hilft da eine visuelle Unterstützung, die dein Vorhaben in den Grundzügen darstellt. Mockups sind digitale Visualisierungen von Ideen und Konzepten oder anders gesagt, eine Art Attrappe des zu gestaltenden Produktes in einer realistischen Umgebung.
Mockups werden zur Visualisierung von Ideen und Konzepten verwendet. Sie werden digital erstellt und dienen zu
Präsentationszwecken
der Absprache mit dem Auftraggeber und den Teammitgliedern
dem Einholen von Feedback

Sie können in ihrem Detailgrad variieren, entsprechen aber meistens schon dem „Look and Feel“ des zu gestaltenden Produktes. Wenn es an die Finalisierung eines Projektes geht, dienen die Mockups häufig als Kommunikationsmittel zwischen Designern und Entwicklern.
Mockups erstellen
Es gibt verschiedene Wege ein Mockup zu erstellen. Der schnellste und einfachste Weg ist die Verwendung von Mockup-Generatoren. Dabei wird das Design einfach als Bild in das Umgebungsmodell hochgeladen. Der Mockup- Generator platziert das Design automatisch und perspektivisch korrekt in das Umgebungsmodell. Anschließend kann das Mockup heruntergeladen und präsentiert werden. Einer dieser Mockup-Generatoren ist Smartmockups. In diesem Video wird erklärt, wie man ihn verwendet. Eine weitere Möglichkeit, Mockups zu erstellen, ist die Verwendung spezieller Software wie z.B. Photoshop. Photoshop ist ein häufig genutztes Werkzeug zur Erstellung von Mockups. CSS Lebenslauf
Verändere deinen Lebenslauf mit CSS. Verwende dazu alle Möglichkeiten, wie du CSS einbinden sowie HTML Elemente verändern kannst mindestens einmal. Gehe dabei wie folgt vor:
Verändere den Hintergrund, indem du zum Beispiel eine Farbe oder ein Bild festlegst.
Die HTML Standard Font ist ziemlich langweilig, suche nach einer modernen Schrift und binde sie in deinen Lebenslauf ein. Verschiedene Fonts findest du unter anderem hier.
Hebe die Überschrift noch mehr hervor, indem du ihr z. B. eine andere Farbe, Größe oder Ähnliches gibst.
Verändere dein Bild, indem du zum Beispiel die Größe oder Border anpasst, Margin oder Padding einfügst.
Vereinheitliche die Farbe oder Größe von kursiv oder fett gedruckten Texten, indem du dessen Selektoren gruppierst und so die entsprechenden Attribute änderst.
Hebe den jeweils ersten Stichpunkt der beiden Listen hervor, indem du dessen Farbe oder Ähnliches änderst.
Verändere mithilfe von Pseudoklassen z. B. die Hintergrundfarbe eines Inputfeldes, wenn der Nutzer darüber „hovert“.
Zuletzt kannst du den Hintergrund vom Footer ändern und den Unterstrich im Link entfernen.

Prettier
Du kannst inzwischen eigene Websites mit HTML und CSS erstellen. Übung gehört dazu, keine Frage. Aber die Grundlagen kannst du nun. Je umfangreicher deine Website und je anspruchsvoller dein Styling wird, umso schneller kann dein Code unübersichtlich werden. Struktur im Code ist das absolute A und O eines erfolgreichen Projektes. Wir möchten an dieser Stelle exemplarisch "Prettier" vorstellen.
Prettier ist eine Extension für deinen Editor, welche dir einen Großteil der Formatierungsarbeit abnimmt. Durch einen Input wird dein Code durchforstet und automatisiert formatiert und nunja, prettier gemacht. Innerhalb der Einstellungen von Prettier findest du alle Anpassungsmöglichkeiten. Speziell die "format on save" Funktion möchten wir dir jedoch sehr ans Herz legen.
